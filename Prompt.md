# AI-Powered Employee Data Management System

## Project Overview

Build a production-ready full-stack AI-Powered Employee Data Management System inspired by modern enterprise HR platforms.

The system must provide:

- Secure employee record management
- Modern animated enterprise dashboard
- Role-based authentication and authorization
- Employee analytics and reporting
- Cloud database integration
- Responsive UI with smooth animations
- Scalable backend architecture
- Production-ready code structure

---

# Context and Role

As a Full-Stack Software Engineer specializing in enterprise applications, your responsibility is to design and develop a secure, scalable, and maintainable Employee Data Management System.

The application must follow modern engineering standards used in real-world enterprise software systems.

The project should emphasize:

- Correctness
- Scalability
- Security
- Clean architecture
- Maintainability
- Performance optimization
- Professional UI/UX
- Clear API structure
- Modular code organization

---

# Primary Objectives

The system must:

- Manage employee records securely
- Provide role-based access control
- Support employee CRUD operations
- Include advanced search and filtering
- Provide analytics dashboards
- Support animated UI interactions
- Store employee documents securely
- Trigger email notifications
- Maintain scalable backend architecture
- Follow secure coding standards
- Be deployment-ready

---

# Required Technology Stack

## Frontend

Use:

- React.js or Next.js
- Tailwind CSS
- Framer Motion
- Axios
- Redux Toolkit or Context API
- React Hook Form
- React Hot Toast
- Recharts or Chart.js

## Backend

Use:

- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- bcrypt
- Nodemailer
- dotenv
- Helmet.js
- Express Rate Limit
- Multer

## Optional Enhancements

- Redis caching
- Docker containerization
- CI/CD pipeline
- AWS deployment
- Socket.io notifications
